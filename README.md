## Card Creation
El proyecto en si seria una una vista de tarjeta de crédito/débito con entradas para poder rellenarlas
de manera con enlace de teclado a través de FocusStatus usando SwiftUI.



### Tecnologías utilizadas:

- Swift
- SwiftUI 
- Xcode 14
